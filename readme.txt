Centralized WordPress widget manager for classic themes with drag & drop ordering, live preview, and full sidebar control.

== Description ==
Wise Widget Manager is a powerful WordPress plugin that provides a centralized dashboard for managing all widget areas (sidebars) in classic WordPress themes.

Instead of navigating multiple widget screens in the WordPress admin, this plugin allows you to control all widgets from a single, unified interface.

It is built for performance, usability, and modern WordPress workflows using REST API and an optimized admin UI.

= Key Features =
- Centralized widget management dashboard
- View all registered widget areas in one place
- Drag & drop widget reordering for better control
- Enable or disable widgets without deleting them
- Remove widgets from sidebars instantly
- Live widget preview in a secure isolated environment
- Fully compatible with classic WordPress widgets
- Lightweight and performance optimized

= How It Works =
1. The plugin detects all registered widget areas (sidebars)
2. It fetches widget data using WordPress widget APIs
3. A centralized dashboard displays all widgets grouped by sidebar
4. Actions like reorder, hide, and remove are handled via REST API


== Installation ==
1. Upload the plugin folder to /wp-content/plugins/
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Tools → Wise Widget Manager in the WordPress admin menu
4. Start managing your widgets

== Frequently Asked Questions ==

= Does this replace the default WordPress widgets screen? =
No. It enhances it by providing a centralized management dashboard.

= Is it compatible with block widgets? =
It is primarily designed for classic widgets but supports most widget structures depending on theme implementation.

= Will it slow down my site? =
No. The plugin is lightweight and uses optimized REST API calls.
